# Importance of Surgical Audit Presentation

This project is an interactive, multi-platform Flutter application that serves as a 12-slide digital presentation on the **Importance of Surgical Audit**.

## Overview
The presentation covers everything from the foundational definition of surgical audits to their crucial role in modern healthcare. It uses humanized, easy-to-understand language and features three real-world clinical examples that demonstrate how audits directly improve patient outcomes, reduce complications, and enhance hospital efficiency.

## Features
- **12 Interactive Slides**: Detailed, human-centric content spanning the entire audit cycle, benefits for stakeholders, challenges, and real-world scenarios.
- **Cross-Platform Responsive Design**: The presentation scales beautifully across Web, Desktop, and Mobile preview screens.
- **Accessible Navigation**: On-screen "Next" and "Previous" controls with clear pagination metrics.
- **Themed UI**: Professional medical blue aesthetics with clear typography and iconography.

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart
- **State Management**: Built-in Stateful widgets with PageController

## Run Instructions
To run this application locally, ensure you have Flutter installed. Then run:

```bash
flutter pub get
flutter run
```
